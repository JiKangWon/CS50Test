8.2 Mảng (Arrays)

8.2.1 Mảng một chiều và hai chiều

Định nghĩa

[cite_start]

Mảng là một tập hợp các phần tử cùng kiểu dữ liệu được lưu trữ dưới một tên duy nhất và được truy cập bằng một chỉ mục (hoặc các chỉ mục). [cite: 195]

Mảng một chiều (1D)

[cite_start]

Khái niệm: một danh sách tuyến tính các phần tử, ví dụ: A[0], A[1], ... A[n-1]. [cite: 197, 201]

[cite_start]

Ví dụ khai báo: [cite: 198]

    [cite_start]
  • Python: arr = [0] * 5 (tạo danh sách có 5 số 0) [cite: 199]
    • [cite_start]
  • Java: int[] arr = new int[5]; [cite: 199]
    • [cite_start]
  • Visual Basic: Dim arr(4) As Integer (chỉ mục 0..4 trong VB.NET) [cite: 200, 202]

Mảng hai chiều (2D)

Khái niệm: một mảng của các mảng; về mặt logic là một lưới hoặc ma trận có hàng và cột. [cite_start]Được truy cập bằng hai chỉ mục: M[row][col]. [cite: 203, 204]

[cite_start]

Ví dụ khai báo: [cite: 205]

    [cite_start]
  • Python: matrix = [[0 for c in range(cols)] for r in range(rows)] [cite: 206, 208]
    • [cite_start]
  • Java: int[][] matrix = new int[rows][cols]; [cite: 207, 209]
    • [cite_start]
  • Visual Basic: Dim matrix(rows-1, cols-1) As Integer [cite: 210]

Bắt đầu chỉ mục: 0 hay 1

[cite_start]

Một số ngôn ngữ sử dụng chỉ mục bắt đầu từ 0 (phổ biến: Python, Java, họ C): phần tử đầu tiên là chỉ mục 0, phần tử cuối cùng là chỉ mục length-1. [cite: 212]

Các môi trường hoặc tài liệu giảng dạy khác đôi khi sử dụng chỉ mục bắt đầu từ 1 (ký hiệu toán học hoặc một số pseudocode/sách giáo khoa). [cite_start]Luôn kiểm tra quy ước của ngôn ngữ. [cite: 213]

Các cạm bẫy phổ biến (Common pitfalls)

    [cite_start]
  • Lỗi Off-by-one: lặp đến <= length thay vì < length gây ra truy cập ngoài giới hạn. [cite: 215]
    • [cite_start]
  • Sai thứ tự chỉ mục cho mảng 2D (nhầm lẫn giữa hàng và cột). [cite: 216]
    • [cite_start]
  • Phần tử chưa được khởi tạo: truy cập các phần tử trước khi chúng được gán giá trị (một số ngôn ngữ cung cấp giá trị mặc định, một số khác thì không). [cite: 217]

for i from 0 to n-1 do
    total = total + arr[i]

Các thao tác 2D phổ biến

    [cite_start]
  • Tổng hàng hoặc cột: sử dụng vòng lặp lồng nhau tính tổng trên chỉ mục thích hợp. [cite: 267, 268]
    • [cite_start]
  • Chuyển vị (Transpose): tạo ra B[c][r] = A[r][c]. [cite: 269]
    • [cite_start]
  • Tìm kiếm: vòng lặp lồng nhau để kiểm tra từng ô nếu cần. [cite: 270]
[cite_start]

Ví dụ - tổng tất cả các phần tử trong mảng 2D (pseudocode): [cite: 271]

total = 0
for r from 0 to rows-1 do
    for c from 0 to cols-1 do

Các thực hành quan trọng

    [cite_start]
  • Luôn đóng tệp sau khi sử dụng để giải phóng tài nguyên và xóa bộ đệm (flush buffers). [cite: 328]
    • [cite_start]
  • Ưu tiên các cấu trúc ngôn ngữ tự động quản lý việc đóng (ví dụ: with của Python, try-with-resources của Java). [cite: 329]
    • [cite_start]
  • Xác thực rằng tệp đã mở thành công và xử lý các ngoại lệ (exceptions). [cite: 330]
    • [cite_start]
  • Cẩn thận với các chế độ (modes): w cắt ngắn (truncates) tệp, a nối vào cuối tệp. [cite: 330]

Ví dụ Python

[cite_start]

Ghi tệp văn bản (an toàn): [cite: 332]

with open('data.txt', 'w', encoding='utf-8') as f:
    f.write('Hello world\n')
    f.write('Second line\n')
# [cite_start]tệp được đóng tự động [cite: 336]
[cite_start]

Đọc tệp văn bản: [cite: 339]

with open('data.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())
[cite_start]

Nối tệp và xử lý lỗi: [cite: 343]

try:
    with open('log.txt', 'a', encoding='utf-8') as log:
        log.write('New entry\n')
except IOError as e:
    print('File error:', e)

Ví dụ Java

[cite_start]

Ghi tệp văn bản (try-with-resources): [cite: 355]

import java.io.*;
try (BufferedWriter writer = new BufferedWriter(new FileWriter("data.txt"))) {
    writer.write("Hello world");
    writer.newLine();
} catch (IOException e) {
    e.printStackTrace();
}
[cite_start]

Đọc tệp văn bản: [cite: 364]

try (BufferedReader reader = new BufferedReader(new FileReader("data.txt"))) {
    String line;
    while ((line = reader.readLine()) != null) {
        System.out.println(line);
    }
} catch (IOException e) {
    e.printStackTrace();
}

Imports System.IO
Using writer As StreamWriter = New StreamWriter("data.txt", False, System.Text.Encoding.UTF8)
    writer.WriteLine("Hello world")
End Using

Đọc tệp văn bản: [cite: 386]

Using reader As StreamReader = New StreamReader("data.txt")
    Dim line As String = reader.ReadLine()
    While line IsNot Nothing
        Console.WriteLine(line)
        line = reader.ReadLine()
    End While
End Using
